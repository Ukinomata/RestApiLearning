2)зарегистрировать handlers
--------
Handler (обработчик) - это функция или метод, который выполняет определенную задачу в ответ на определенное событие или запрос. Например,
веб-обработчик (web handler) - это функция, которая обрабатывает запросы клиента на веб-сервере и возвращает соответствующие ответы.
Обработчик событий (event handler) - это метод, который выполняет определенные действия в ответ на определенное событие,
например, щелчок на кнопке мыши. Обработчики используются в различных программах и системах для обработки событий и запросов пользователей.

Handler должен иметь определенную сигнатуру:

--
func ExampleHandler(w http.ResponseWriter, r *http.Request, params httprouter.Params) {

}
можно без параметров(params) если мы хотим совместимость с дефолтным http роутером.

в таком случае нужно вместо
router.GET использоваться router.HandlerFunc
--
